JS插件

概述

Lion-UI拥有强大且丰富的组件,提供给后台开发一整套控件选择,大大提升了开发速度。

框架以简洁高效为追求,实现完美UI体验。

下拉框控件

属性

名称 描述
id 规定该标签的唯一id。
name 规定该标签的名称。
placeholder 规定如果该标签的值为空时，显示的内容。
class 规定该标签的样式。推荐样式（”tdon-combo form-control select2”）
data-valueField 规定该标签下拉框每一个元素的值所对应的对象的属性。
data-textField 规定该标签下拉框每一个元素显示的值所对应的对象的属性。
data-URL 规定该标签获取数据的url。
